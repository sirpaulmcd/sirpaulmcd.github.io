# Introduction
There are 5 fundamental behavioural patterns
  - Command
  - Iterator
  - Observer
  - State
  - Template method

Additional behavioural patterns:
  - Chain of responsibility
  - Interpreter
  - Mediator
  - Memento
  - Visitor
